Notes and comments about the migration of Arsnova from Sencha Touch 1.1 to Sencha Touch 2.1.

---------
Notes
---------

If using panel.getValues() the items holding the forms must have a name attribute, otherwise
this function will return an empty object.

var form = Ext.create('Ext.form.FormPanel', {
	...
	items: [{
		xtype: 'fieldset',
		items: [{
			xtype: 'textarea',	
			name: 'first'		// important
		}, {
			xtype: 'textarea',
			name: 'second'		// important
		}
	}
}

var values = form.getValues();
console.log(values.first);
console.log(values.second);


---------
Warnings:
---------

Warning concerning scrollable: false and heigth is a sencha touch bug:
http://www.sencha.com/forum/showthread.php?256858-Warning-when-scrollable-is-false-but-height-is-specified

Workaround:
Set the scrollable attribute as follows deactivates the scrolling and let the warnings disappear: 
	
	scrollable: { disabled: true }


---------
Errors:
---------

Lists are not shown properly. Without using vbox layout on the container and the flex property on
the list you even can not make the list visible. When you use this config the list will fit automatically
to the screen. So if your device or screen is not big enough to show the whole list, some list items may
disappear - even if you scroll the panel. One possibility would be to make the list scrollable, but with
this config it would not match the arsnova-legacy project. I browsed the web and found this workaround:

http://www.sencha.com/forum/showthread.php?248193-2.1-Lists-within-a-container&p=934761&viewfull=1#post934761

You add a initialize function to the listener, which sets the true height of the list. With this workaround
you don't need any vbox, fit or flex layouts anymore to show the list properly.

Workaround:
	var list = Ext.create('Ext.List', {
		...
		listeners: {
		    initialize: function (list, eOpts){
		        var me = this;
		        if (typeof me.getItemMap == 'function'){
		            me.getScrollable().getScroller().on('refresh',function(scroller,eOpts){
		                me.setHeight(me.getItemMap().getTotalHeight()+20);
		            });
		        }
		    }
		}
	});
